# 💸 AI Expense Tracker (SpendAI)

An AI-powered expense tracking web application built using Flask.

## 🚀 Features

* Add & manage expenses
* AI-based automatic categorization
* Dashboard analytics
* Monthly tracking

## 🛠 Tech Stack

* Python (Flask)
* HTML, CSS, JavaScript
* SQLite

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open: http://127.0.0.1:5000
